# Sudoku Solver
A simple terminal application that solves a Sudoku puzzle.

## How to use
### 1. compile
```
$ make
```

### 2. execute
```
$ ./sudokusolver
```

### 3. input the grid

example:
```
Input:
   123456789

1    9  7 4
2   71 2   5
3   4     39
4       8
5     46
6    219 8
7   6    4
8   9 2865
9  5
```
